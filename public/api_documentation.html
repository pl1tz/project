API Documentation

Cars

GET /cars?page=1 - Список автомобилей

GET /cars?coll=all - Список автомобилей без пагинации

GET /car_ids?id=30 - Список автомобилей для редактирования по id

GET /cars/:id - Показать автомобиль

GET /cars?newest=true - Список автомобилей новых

GET /cars?mileage=true - Список автомобилей по пробегу

GET /cars?price_desc=true - Список автомобилей по убыванию цены

GET /cars?price_asc=true - Список автомобилей по возрастанию цены

GET /car_details?brand_name=(НАЗВАНИЕ БРЕНДА)&model_name=(НАЗВАНИЕ МОДЕЛИ)&generation_name=(НАЗВАНИЕ ПОКОЛЕНИЯ) - Детали автомобилей про бренду, модели и поколения

GET /filters - Фильтры для автомобилей(фильтры принимаются такие же как и на список автомобилей)

            {
                "brand_name": "Ford",
                "model_name": "Focus",
                "generation_name": "III Рестайлинг",
                "year_from": 2015,
                "max_price": 30000,
                "gearbox_type_name": "Автоматическая",
                "body_type_name": "Седан",
                "drive_type_name": "Полный",
                "owners_count": 1,
                "engine_name_type_name": "Дизель"
            }

Фильтры для автомобилей

  • brand_name - Название бренда автомобиля
  • model_name - Название модели автомобиля
  • generation_name - Название поколения автомобиля
  • year_from - Год выпуска, начиная с
  • max_price - Максимальная цена
  • gearbox_type_name - Тип коробки передач
  • body_type_name - Тип кузова
  • drive_type_name - Тип привода
  • owners_count - Количество владельцев
  • engine_name_type_name - Тип двигателя
